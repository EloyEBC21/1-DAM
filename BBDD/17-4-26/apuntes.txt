Consulta de datos

A. Sentencia SELECT
SELECT [ALL | DISTINCTI]
[expre_colum1,expre_colum2,...,exprecolumn[*]]
FROM [nombre_tabla, nombre_tabla2; ..., nombre_tablan]
[WHERE condicion]
[ORDER BY expre_column [DESC|ASC] [,expre_colum [DESC|ASC]]...];

B. Clausulas de SELECT
*   FROM: FROM [nombre_tabla1, nombre_tabla2, ..., nombre_tablan]
*   WHERE: WHRE [WHERE condición]
    Ej:
        WHERE NOTA=5
        WHERE (NOTA>=10) AND (CURSO=1)
        WHERE (NOTA IS NULL) OR (UPPER(NOM_ALUMN) = 'PEDRO')
*   ORDER BY: [ORDER BY expre_columna [DESC | ASC][, EXPRE_COLUMNA [DESC|ASC]]...]
        Ej:
        SELECT * FROM ALUMNOS ORDER BY NOTA/5;
        SELECT * FROM ALUMNOS ORDER BY NOM_AULMN, CURSO DESC;
        SELECT DEPT_NO, DNOMBRE, LOC FROM DEPART ORDER BY 2;
*   ALL: Recuperar todas las filas, aunque algunas estén repeidas.
*   DISTINCT: Sólo recupera las filas que son distintas.
        Ej:
        Columna DEPT_NO -> TABLA EMPLE

C. Seleccion de columnas
SELECT [ALL|DISTINCT]
    [expre_column1, expre_column2, ..., expre_column[*]]
FROM [nombre_tabla1,nombre_tabla2, ..., nombre_tablan]
